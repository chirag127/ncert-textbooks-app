# oriz-books

📚 NCERT textbook directory — browse, search, and download for free.

Part of the [oriz](https://github.com/chirag127/oriz) family of static sites.
Live at <https://books.oriz.in>.

## What it does

- Browse the full NCERT textbook catalogue by class, subject, and language.
- Search across titles, subjects, chapters, and keywords.
- Download individual chapters or the merged whole-book PDF — links go directly
  to <https://ncert.nic.in>; nothing is hosted on this domain.
- Optional sign-in (shared with the rest of `*.oriz.in` via Firebase Auth) to
  bookmark books.

## Tech

- **Astro 6** static site, **React 19** islands, **Tailwind v4**.
- **[@chirag127/oriz-ui](https://github.com/chirag127/oriz-ui)** for the design
  system, Sidebar, AccountPanel, FinishSignIn, ContactForm, and FAMILY_SITES.
- **Firebase Auth + Firestore** (shared `oriz-app` project).
- **Cloudflare Pages** for hosting via `wrangler deploy`.

## Develop

```bash
pnpm install
npx envpact-cli@0.2.0          # writes .env from the shared envpact bundle
pnpm dev                       # http://localhost:4321
```

The book grid and detail pages read from `public/data/metadata/books.json` at
runtime (and the `[bookCode].astro` route reads it again at build time via
`src/lib/books.ts` to generate static paths).

## Build + deploy

```bash
pnpm build                     # outputs to ./dist
pnpm deploy                    # wrangler deploy (Cloudflare Pages)
```

`books.oriz.in` is bound to the `oriz-books` Cloudflare Pages project from the
dashboard.

## Data pipeline

The `books.json` and per-class/subject/language indexes under
`public/data/metadata/` are generated by a Playwright scraper that hits
`ncert.nic.in`. The scraper source lives outside this repo (slated to move to
`scripts/scraper/` in a future commit). Today, the JSON shipped here is a
snapshot — refresh by re-running the scraper and overwriting these files.

`tools/python/` contains PyPDF2 utilities for ad-hoc inspection of downloaded
PDFs; not part of the website build.

## Layout

```
src/
  components/      Header, Footer, Sidebar wrapper, BookGrid, BookDetail, BookSearch, HeaderControls
  layouts/         BaseLayout.astro
  lib/             siteConfig, firebase, books loader, types, preflight script
  pages/           index, search, about, contact, account, account/finish-sign-in, book/[bookCode], legal/*
  styles/          global.css (just imports tailwind + oriz-ui styles)
public/
  data/metadata/   books.json + per-class/subject/language/bookcode indexes
  data/search-index/ documents.json (search seed)
  favicon.svg
data/              source-of-truth copy of metadata; copied into public/ on each build snapshot
tools/python/      PyPDF2 helpers
```
